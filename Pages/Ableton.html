Max4Live Devices

KOUTPUT

An audio aggregator and merger built for Ableton Live using Max4Live.

Download via GitHub.

It's not very user-friendly. The intended use case is quite specific and requires external audio routing. On macOS, I use Existential Audio's Black Hole configured to have 4 channels to handle the main and cue channels.
I use Koutput to capture system audio and distribute it to multiple output sources with individual processing on each channel with volume control. Note: audio processing has to be done in the Max4Live patcher.
